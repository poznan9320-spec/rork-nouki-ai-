# Delivery Hub — 物流・営業用 iOS アプリ


## 概要
EMERENT バックエンドと完全連携した、現場・屋外向けの物流・営業用 iOS ネイティブアプリを構築します。

---

## Features（機能）

**ホーム（入荷予定）**
- 今日 / 明日 / 1週間 のセグメントコントロールで入荷リストをフィルタリング
- 各入荷カードに商品名・数量・メーカー・ステータス・納品日を表示
- ステータスバッジ（PENDING / 完了 など）をカラーで視覚的に区別
- プルリフレッシュで最新データを再取得

**AI チャット検索（営業向け）**
- テキスト入力で EMERENT AI に在庫・入荷状況を問い合わせ
- 回答を「大きな1枚の回答カード」で画面中央に大きく表示（客先で見せやすい）
- 過去の検索履歴を下部にリスト表示

**発注依頼フォーム**
- 商品名・数量・メモを入力して発注依頼を送信
- カメラで現場写真を撮影し Base64 で添付可能
- 送信完了トースト通知

**共通・インフラ**
- 全通信に `X-Company-ID: c063417b` ヘッダーを自動付与
- ネットワークエラー時はわかりやすい日本語メッセージを表示
- ローディング中はスケルトンカードでプレースホルダー表示

---

## Design（デザイン）

- **テーマ**: 現場・屋外での視認性最優先の「ハイ・コントラスト UI」
- **カラー**: ダークネイビー背景 (`#0D1B2A`) + 鮮やかなブルー/ティール アクセント、白文字
- **タイポグラフィ**: SF Pro 太字・大きめフォントサイズで視認性アップ
- **カード**: 角丸・深めのシャドウ、ステータスは左ボーダーのカラーラインで一目でわかるデザイン
- **回答カード（Searchタブ）**: 画面の大部分を占める巨大な白カード、回答テキストは 28pt 以上の太字
- **アニメーション**: カード出現時のスライドイン、ローディング時のパルスアニメーション
- **アイコン**: 物流・配送をイメージしたトラック/箱モチーフのアプリアイコン（ダークネイビー + ブルーグラデーション）

---

## Screens（画面構成）

| タブ | 内容 |
|------|------|
| 🏠 Home | セグメント切り替え付き入荷予定リスト。カードをタップで詳細表示 |
| 🔍 Search | AIチャット検索。大きな入力バーと巨大な回答カード |
| 📋 Request | 発注依頼フォーム（商品名・数量・メモ・写真添付） |
